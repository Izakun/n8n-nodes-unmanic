# n8n-nodes-unmanic

[![npm version](https://img.shields.io/npm/v/n8n-nodes-unmanic.svg)](https://www.npmjs.com/package/n8n-nodes-unmanic)

n8n community node for Unmanic (library optimizer) via its API

Install via **Settings -> Community Nodes -> Install** -> `n8n-nodes-unmanic`.

## Usage example

Read the version:

1. Add the node after a trigger (e.g. *When clicking 'Test workflow'*).
2. Select your credential.
3. **Get Version**.
4. Execute the node — example output:

```json
{ "version": "0.2.5" }
```
